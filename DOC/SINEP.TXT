                     1. ОБЩИЕ ПОЛОЖЕНИЯ

     Программа SINEP предназначена для выделения единичных
вызванных потенциалов в электроэнцефалограмме. В состав
программы входят исполняемый файл SINEP.EXE и текстовый файл
параметров программы SINEP.CFG. Программа написана на языке
С++ и функционирует под управлением операционной среды MS
DOS версии 3.0 и старше на ПЭВМ типа IBM PC AT, снабженной
сопроцессором с плавающей точкой. Для работы программы
необходимо присутствие в одном каталоге с программой файла
конфигурации SINEP.CFG и, если используется графический
вывод, графического драйвера EGAVGA.BGI в доступном по
умолчанию каталоге.



                    3. РАБОТА ПРОГРАММЫ

                  3.1 Запуск и управление

     Программа SINEP запускается командной строкой

sinep.exe name.inf

, где name.inf имя информационного файла, содержащего
описание структуры данных и перечень файлов с данными.
Подробнее об информационном файле и структуре данных см.
описание к программе EEGIN.
     После запуска, программа запрашивает имя
информационного файла, в котором будут описаны выходные
данные. По умолчанию берется имя входного информационного
файла с расширением .fin.
     Прервать работу программы можно нажатием клавиши Esc.


               3.2 Входные и выходные данные
                              
     Входными данными программы служат оцифрованные отрезки
электроэнцефаллограммы, содержащие реализации вызванных
потенциалов. Для выделения ВП по методу комплексной
демодуляции вызванному потенциалу должен предшествовать
отрезок фоновой ЭЭГ, а после вызванного потенциала должен
иметься отрезок последействия. Все три отрезка должны иметь
сопоставимый размер. Формат данных может быть целый
беззнаковый со смещением 2048 (формат АЦП L-205), байтовый
знаковый или символьный. Данные располагаются поканально в
виде матрицы, в которой номер столбца соответствует номеру
канала.
     Выходные данные имеют символьный формат и расширение
имени .sep.
     Если в файле конфигурации задан параметр анализа фазы
фона, на которую пришелся стимул, то в выходном
информационном файле для каждого файла данных и каждого ЭЭГ
канала будет указана приведенная к косинусу фаза в радианах.
Т.е. нулевая фаза будет соответствовать максимуму фоновой
молны; +пи и -пи - минимуму; отрицательная фаза -
восходящему фронту; положительная - нисходящему.


              3.3 Структура файла конфигурации
                              
     Файл конфигурации программы SINEP.CFG содержит
параметры программы и, возможно, параметры описывающие
входные данные. Параметры файла конфигурации могут повторять
описатели данных из информационного файла входных данных, в
этом случае соответствующие параметры файла конфигурации
игнорируются.
     Формат параметра из файла конфигации:

name=value

, где name  - имя параметра
      value - его значение.
Ниже следует список и описание возможных параметров
программы файла конфигурации.

     label - номер отсчета в данных, соответствующий моменту
стимула;

     first_ep - номер отсчета в данных, с которого
начинается выделяемый ВП или его компонент;

     last_ep - номер отсчета в данных, которым заканчивается
выделяемый ВП или его компонент;

     ca - коэффициент усиления данных. Этот параметр на
работу программы непосредственного влияния не оказывает, но
позволяет оценить амплитуду ВП во время его графической
визуализации;

     min_frq - минимальная частота, входящая в ВП в Гц;

     max_frq - максимальная частота, входящая в ВП в Гц;

     ord_frq - анализируемая частота фона в Гц;
     level - уровень визуализации при работе программы.
Может быть 0, 1, 2 или 3. При level=0 графический вывод не
используется;

     demodulation - использовать или для выделения ВП метод
комплексной демодуляции. Возможные значения параметра Yes
или No;

     fourie - использовать или для выделения ВП метод Фурье
фильтрации. Возможные значения параметра Yes или No;

     order - порядок интерполяционного многочлена для метода
комплексной демодуляции. Определяет степень удаления фоновой
ЭЭГ из ВП. Увеличение порядка ведет к снижению амплитуды ВП
и расходу памяти.

     coef_mul - коэффициент увеличения сигнала. Параметр
передается в выходной информационный файл и используется
программой EEGIN при необходимости просмотра выходных данных
в графическом виде.
